
# **Authentification Web avec CGI en C++**

## Description
Ce projet met en œuvre un système d'authentification pour les applications web en utilisant CGI (Common Gateway Interface) avec le langage C++. Le programme traite les données du formulaire de connexion en utilisant les méthodes GET et POST, vérifie les informations d'identification contre un fichier de stockage, et affiche des pages de réponse appropriées. Le système assure que les utilisateurs reçoivent une rétroaction claire en cas d'erreurs, tout en conservant les valeurs saisies dans le formulaire. Ce projet illustre l'intégration des formulaires HTML avec le traitement côté serveur en C++ pour une gestion sécurisée des connexions utilisateur.
